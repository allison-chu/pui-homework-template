What is NASA's MoonRanger?

Worked on the user interface for NASA's MoonRanger mission. Operators may only have 2 hours twice a day during the 14 day mission, so an intuitive user interface is necessary for efficient task completion.

Problem

The existing interface was outdated and not intuitive to users, requiring significant training time and reducing operational efficiency during critical mission windows.

Solution

Met with the engineering team to discuss the redesign, focusing on:

  • Streamlined navigation
  • Intuitive controls
  • Modern interface design
